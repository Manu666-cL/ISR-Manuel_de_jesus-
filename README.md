# Calculadora de ISR 2026 - República Dominicana

## Resumen
Este programa en C++ permite calcular el Impuesto Sobre la Renta (ISR) mensual para un empleado, aplicando primero el descuento obligatorio del 5.91% de la TSS, basado en las escalas salariales vigentes para el año 2026.
